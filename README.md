# UIKA: Fast Universal Head Avatar from Pose-Free Images

[[Paper](https://arxiv.org/abs/2601.07603)] | [[Homepage](https://zijian-wu.github.io/uika-page/)]

<div align=center>
  <img src="./assets/uika_teaser.gif">
</div>

> We present UIKA, a feed-forward animatable Gaussian head model from an arbitrary number of pose-free inputs, including a single image, multi-view captures, and smartphone-captured videos.

## Release Plan

We are in our company's internal review and compliance process for open-sourcing this project. We plan to release the following components before the end of April 2026:
